# SQL Business Analytics Bootcamp

##  Overview

This repository demonstrates my SQL learning journey and hands-on practice in Business Analytics using MySQL.

The project covers database creation, data insertion, SQL querying, aggregation, and relational database joins.

##  Project Structure

schema.sql

Creates the database and tables.

data.sql

Inserts sample customer and order data.

basic_queries.sql

Contains SQL queries for filtering, sorting, grouping and aggregation.

joins.sql

Demonstrates INNER JOIN, LEFT JOIN and RIGHT JOIN.
